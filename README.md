# Gerenciamento Financeiro (Liquid Glass)

Projeto com UI Liquid Glass, Firebase Auth + Firestore, Donut Chart e exportação PDF/CSV.

## Como usar

1. Copie `.env` (já preenchido) ou verifique que as variáveis estão corretas.
2. Instale dependências:
```
npm install
```
3. Rode em desenvolvimento:
```
npm run dev
```


Notas: selects agora são estilizados .input-glass; categorias adicionadas via modal.
